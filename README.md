# piSensorLib


## Microduino树莓派扩展板传感器库


### 硬件提供接口：
+ ADC：ADC1, ADC2, ADC3, ADC4, 
+ I2C:
+ UART:
+ GPIO:IO12, IO13, IO16,
+ TFT:
+ I2S:


### 库安装：
+ LX终端,进入文件夹：输入 sudo sh ./install.sh

### 卸载：
+ LX终端,进入文件夹：输入 sudo sh ./uninstall.sh


### 示例：
+ adcTest.py  ADC测试
+ adcControlServo.py  ADC显示,舵机测试
+ gpioTest.py   GPIO测试
+ lm75Test.py  LM75测试
+ servoTest.py  舵机测试
+ tempHumi.py   温湿度测试
+ dotMatrixTest.py  点阵测试
